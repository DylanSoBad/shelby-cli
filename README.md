# Shelby CLI

Lightweight command line interface for interacting with the Shelby decentralized storage network.

Shelby CLI allows developers to upload files and retrieve blobs directly from the terminal with a simple workflow.

---

## Features

• Upload files to Shelby storage  
• Fetch stored blobs by ID  
• Simple CLI interface  
• Lightweight developer tool  
• Easy integration with scripts and automation
